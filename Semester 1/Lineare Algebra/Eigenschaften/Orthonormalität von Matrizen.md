Eine orthogonale Matrix ist in der [[Übersicht - lineare Algebra|linearen Algebra]] eine [[quadratische]], [[reelle]] Matrix, deren Zeilen- und Spaltenvektoren orthonormal bezüglich des [[Grundlegende Matrix Operationen#Multiplikation von Matrizen|Standardskalarprodukts]] sind. Damit ist die [[Grundlegende Matrix Operationen#Inverse einer Matrix|Inverse]] einer orthogonalen Matrix gleichzeitig ihre [[Grundlegende Matrix Operationen#Transponieren von Matrizen|Transponierte]].
