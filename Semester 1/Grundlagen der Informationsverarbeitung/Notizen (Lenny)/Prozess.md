# Definition

Ein Prozess ist der Vorgang einer algorithmisch ablaufenden Informationsverarbeitung.

# Beschreibung

- Vorgang innerhalb des [[Betriebssystem|Betriebssystems]] welches durch eine [[Programm]] gesteuert wird
- Eigenschaften:
	- Laufen geordnet ab, heißt, sie werden gestartet, überwacht und beendet
	- Können unterbrochen werden und zu späteren Zeitpunkt fortgesetzt werden
	- Kann durch andere [[Prozess|Prozesse]] gesteuert werden
	- Werden durch den [[Prozessor]] ausgeführt
	- Können gleichzeitig oder nacheinander ablaufen
	- Mehrere [[Prozess|Prozesse]] können in einen [[Prozess]] zusammengefasst werden
	- [[Prozess|Prozesse]] können [[fork|aufgespalten (fork)]] und wieder [[join|zusammengeführt (join)]] werden
