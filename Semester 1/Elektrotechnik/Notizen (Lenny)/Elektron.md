# Geschwindigkeit im [[Leiter]]

[[Elektronendichte]]: $n = 8.47 * 10^{19} \dfrac{1}{mm^3}$ für Kupfer

[[Ladung]]:
$Q = I * t$
$I = e \text{ (Elementarladung) } * n \text{ (Elektronendichte) } \Rightarrow \text{ Elemtarladung mal die Anzahl der Elektronen}$
$Q = e * n * A * l$
$I = \dfrac{Q}{t} = \dfrac{e * n * A * l}{t} = e * n * A * v$
$v = \dfrac{I}{e * n * A}$
Ergebnis im Bereich $\dfrac{mm}{s}$
