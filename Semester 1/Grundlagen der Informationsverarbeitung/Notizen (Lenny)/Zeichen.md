# Definition

Ein Zeichen ist ein Symbol aus einer Menge von Symbolen, die zur Darstellung von [[Informationen]] dienen

# Aufteilung

- numerische [[Zeichen]] sind die Ziffern von 0 bis 9
- alphabetische Zeichen sind die Buchstaben
- Sonderzeichen, wie + # * & % $ usw.
