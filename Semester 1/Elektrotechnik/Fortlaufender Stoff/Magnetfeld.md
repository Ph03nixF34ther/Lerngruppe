# Spulen zur Erzeugung von Magnetfeldern
## Zylinderspule
### Beschreibung:
- langer aufgewickelter  Draht
- Ergibt die Form eines Zylinders

// Abb. Spule

### Charakteristika

| Formelzeichen | Interessante Charakteristika         | Verlinkung |
| ------------- | ------------------------------------ | ---------- |
| $L$           | axiale Ausdehnung der Wickelung      | [[Länge]]  |
| $r$           | Radius / Durchmesser der Wicklung    |            |
| $N$           | Windungszahl - Anzahl der Wicklungen |            |

ggf: Windungsdicht ($\frac{N}{L}$) Anzahl der Windungen pro Längeneinheit

![[Magnetfeld_Spule_Zeichnung.jpeg]]

### Annahmen für das idealisierte Feld:
- L >> r
- gleichmäßige Wicklung entlang der Spulenachse
- dünner Draht => enge Wicklung
- kein Kern, oder Homogener Kern
=> Randeffekte können vernachlässigt werden

## Ring- oder Toroidspulen
Beschreibung: 
- ringförmiger Kern (Luft // ferromagnetisches Material)
- Wicklung, die den Kern in Umlaufsrichtung mehrfach umschließt

- magnetische Felder der einzelnen Windungen addieren sich im Kern
- außerhalb sind die Beiträge weitgehend entgegengesetzt => Auslöschung
=> starkes, geschlossenes Feld im Kern der Toroidspule, Magnetfeld außerhalb verspindet
### Charakteristika

| Formelzeichen                                        | Interessante Charakteristika         | Verlinkung            |
| ---------------------------------------------------- | ------------------------------------ | --------------------- |
| $r_{m} = \frac{r_{i} + r_{a}}{2}$                    | axiale Ausdehnung der Wickelung      | [[Länge]]             |
| $A = \left( \frac{r_{a} - r_{i}}{2} \right)^2 * \pi$ | Radius / Durchmesser der Wicklung    | [[Querschnittsebene]] |
| $N$                                                  | Windungszahl - Anzahl der Wicklungen |                       |

### Annahmen für das idealisierte Feld:
- unendlich viele Windungen, gleichmäßig über den Kern verteilt
- dünner Kern. $r_{a} - r_{i} < r_{m}$
- keine Streufelder außerhalb des Kerns
- homogenes Material im Kern
=> Randeffekte können vernachlässigt werden
